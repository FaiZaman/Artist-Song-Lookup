To run on Windows on the University machines:
 
Run Python 3.7 from the app hub first (the bash files do not work on university machines)
Navigate to the directory of the python files and type "python3 server.py" or "python3 client.py" to run server and client respectively

To run on Linux on University machines:

Navigate to directory of python files and run server by typing "python 3 server.py" and then client by "python3 client.py"

My bash files work on my machine but not on the universty Windows. They may work on your personal machine.
